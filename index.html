My Tools

I have a strong track record of developing innovative tools such as for'CE, DiagonAlley, RoBoT-A, CertImporter, Agent Monitor, and Sumo Ready, and many more which have significantly enhanced debugging capabilities and customer support efficiency.

For'CE

The centralized AI log analyzer efficiently collects and analyzes logs, providing comprehensive solutions that unlock the potential of logs.

DiagonAlley

DiagonAlley is a single-page application (SPA) that offers a user-friendly interface for collecting logs, significantly reducing the time required to obtain logs from customers.

RoBoT-A

A browser extension that assist support engineers by automating access to IICS organizations, retrieving logs, and integrating with Jira, kb, alerts, and timers from SFDC.

CertImporter

Java-based tool automates the process of importing SSL certificates into the JRE's cacerts file, streamlining certificate management. The tool establishes SSL connections to a specified host, retrieves the SSL certificates, and imports them into the cacerts file, which is used by Java applications for trust verification.

Agent Monitor

This Java tool leverages Informatica Cloud Public REST APIs to monitor Informatica Cloud Secure Agents. It uses Informatica Cloud credentials to log in, checks the status of the agents, and sends email alerts if any agent is not in a RUNNING state. Additionally, it can automatically restart any agent service that is found to be in an ERROR state.

KnowYourETL & ReFiner

KnowYourETL tool indexes and searches IICS assets, providing detailed insights into tasks, connections, and properties. ReFiner facilitates efficient management of IICS assets by allowing users to filter, update in bulk, and manage connection properties like passwords, hosts, and more.
